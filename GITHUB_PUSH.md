# Push to GitHub

1. Create a new repository on GitHub: https://github.com/new  
   - Name: `elementor-agency-affiliate-page`  
   - Public, no README (already committed)

2. Add remote and push:
```bash
git remote add origin https://github.com/YOUR_USERNAME/elementor-agency-affiliate-page.git
git push -u origin main
```
Replace `YOUR_USERNAME` with your GitHub username.
