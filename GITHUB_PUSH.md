# Push to GitHub

1. Create repo: https://github.com/new?name=elementor-agency-affiliate-page  
   (Public, **do NOT** add README – code đã có sẵn)

2. Chạy:
```bash
git remote add origin https://github.com/YOUR_USERNAME/elementor-agency-affiliate-page.git
git push -u origin main
```
Thay `YOUR_USERNAME` bằng username GitHub của bạn.
